# whatsapp筛号 隐私政策

最后更新：2026-06-09

## 一、概述

whatsapp筛号 是一个 Chrome 扩展，用于帮助企业用户在 WhatsApp Web 中批量筛查号码，并判断号码是否看起来已注册 WhatsApp。

## 二、我们会处理哪些数据

当你使用本扩展时，扩展可能会处理以下数据：

- 你手动粘贴或导入的手机号/号码列表
- 当前页面会话中的筛号结果，例如某个号码是否看起来已注册 WhatsApp

## 三、这些数据的用途

这些数据仅用于：

- 在你当前打开的 WhatsApp Web 会话中执行号码筛查
- 在扩展界面中展示当前会话的筛号进度与结果
- 让你将当前结果导出为 CSV 或 Excel 文件

## 四、数据存储方式

- 当前上架版本不提供本地历史记录持久化功能
- 筛号结果默认仅保留在当前页面会话中；刷新页面、关闭页面或关闭扩展后不会继续保存
- 当前实现中，扩展不会主动将你的号码列表、筛号结果上传到开发者自有服务器

## 五、数据共享说明

- 我们不会主动出售、出租或向第三方共享你的号码列表或筛号结果
- 但由于本扩展运行在 WhatsApp Web 页面内，你在使用 WhatsApp Web 服务时，相关请求和数据仍可能按照 WhatsApp 自身的服务机制被处理

## 六、权限与访问说明

本扩展当前仅请求以下能力，以实现核心功能：

- `activeTab`：用于从扩展入口打开当前使用页面
- `downloads`：用于导出 CSV 或 Excel 文件到你的设备
- `https://web.whatsapp.com/*` 页面访问权限：用于在 WhatsApp Web 页面中执行筛号功能
