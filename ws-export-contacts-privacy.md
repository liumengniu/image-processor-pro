# Privacy Policy

## WhatsApp Export Contacts Privacy Policy

### Last Updated: June 12, 2026

Thank you for using WhatsApp Export Contacts extension. This privacy policy explains how we collect, use, and protect your information.

---

## Information Collection and Use

### What We Don't Collect

WhatsApp Export Contacts extension **does NOT collect, store, or upload** any personal information or user data, including but not limited to:

- WhatsApp account information
- Contact list
- Chat history
- Group information
- Browsing history
- Device information

### Local Processing

All features are executed locally in your browser:

1. **Contact Export**: Exported contact data is only used to generate Excel file downloads
2. **Group Members**: Retrieved group member information is only used for export to local Excel
3. **Group List**: Group information is only used for export to local Excel
4. **Operation Logs**: Operation status is only displayed in the extension interface, not uploaded or saved

---

## Data Flow

```
User Operation → Browser Local Processing → Generate Excel File Download
              ↓
         No Server Storage
```

---

## Permission Usage

This extension requests the following permissions:

| Permission | Purpose | Description |
|------------|---------|-------------|
| `activeTab` | Access current tab | Only activates when user clicks the extension icon, used to get contact data in WhatsApp Web |
| `host_permissions: web.whatsapp.com` | Host permission | Only allows the extension to run on WhatsApp Web pages |
| `tabs` | Create tabs | When user is not on WhatsApp Web, automatically opens web.whatsapp.com in a new tab |

**We will NOT**:
- Access other websites
- Read or modify webpage content (except WhatsApp Web)
- Monitor system activities
- Install other software
- Upload any data to servers

---

## Third-Party Services

This extension:
- Does not use any third-party analytics services
- Does not contain any trackers or advertisements
- Does not communicate with any external servers (except WhatsApp Web itself)

---

## Data Security

Since we do not collect any data, there is no risk of data leakage or misuse. All operations are completed locally on your device. Exported Excel files are kept by you.
