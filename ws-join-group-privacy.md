# 隐私政策

## WhatsApp批量进群 隐私政策

### 最后更新日期：2026年6月11日

感谢您使用 WhatsApp批量进群 扩展程序。本隐私政策旨在帮助您了解我们如何收集、使用和保护您的信息。

---

## 信息收集与使用

### 我们不收集的内容

WhatsApp批量进群 扩展程序 **不会收集、存储或上传** 任何个人信息或用户数据，包括但不限于：

- WhatsApp 账号信息
- 联系人列表
- 聊天记录
- 群组信息
- 浏览历史
- 设备信息

### 本地处理

所有功能均在您的本地浏览器中执行：

1. **群链接处理**：您输入的群邀请链接仅用于本地执行加入群组操作
2. **时间间隔**：间隔设置存储在浏览器本地，不会同步到任何服务器
3. **操作日志**：日志仅显示在扩展界面，不会上传或保存

---

## 数据流向

```
用户输入 → 浏览器本地处理 → WhatsApp Web API
              ↓
         无任何服务器存储
```

---

## 权限使用说明

本扩展请求以下权限：

| 权限 | 用途 | 说明 |
|------|------|------|
| `activeTab` | 访问当前标签页 | 仅在用户点击扩展图标时激活，用于在 WhatsApp Web 中显示操作面板 |
| `host_permissions: web.whatsapp.com` | 主机权限 | 仅允许扩展在 WhatsApp Web 页面上运行 |

**我们不会**：
- 访问其他网站
- 读取或修改网页内容（除 WhatsApp Web）
- 监控系统活动
- 安装其他软件

---

## 第三方服务

本扩展程序：
- 不使用任何第三方分析服务
- 不包含任何追踪器或广告
- 不与任何外部服务器通信（除 WhatsApp Web 本身）

---

## 数据安全

由于我们不收集任何数据，因此不存在数据泄露或滥用的风险。所有操作均在您的设备本地完成。
